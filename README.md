# 🧭 一人公司罗盘

为 AI 时代的超级个体打造，帮助你测算收入目标、产品定价、销量需求与盈亏平衡点。

## 功能

- **📊 一人公司适合度测评** — 11 维度雷达图，看清你的优势和短板
- **💰 ROI 利润分析** — 产品型 / 服务型双模式，盈亏平衡 + 利润诊断
- **📈 目标利润推导** — 设定目标，反算所需销量或时薪
- **🤖 AI 深度报告** — 根据测评结果生成个性化分析
- **🌐 中英双语** — 支持简体中文 / English 切换

## 技术栈

微信小程序 · Taro + React · TypeScript · 云开发
