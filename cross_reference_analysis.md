## 二、新增资料的增量价值

### C：「你适合创业吗」PDF

- 提供经过验证的 10 维度自评框架（A/B 栏对比二选一，A/B 各自汇总）
- 评分逻辑：每维度 A 得分 0–10（6–10 强项，0–4 不太强），B 得分 0–10（6–10 弱项）；A 总分 ≥ 50/100 适合创业
- **应用**：作为测评表的主体框架，融入 OPC 内容 + 新增 AI 技能第 11 维度；四档分类是在原始两档基础上的增强

### D：「如何开启你的一人公司」

- OPC 运营的实操指南（扫描件，无法 OCR）
- 主题方向补充了对一人公司模式的认知
- **应用**：融入测评各维度的 OPC 情景题

### E：「用代码和自媒体杠杆，做 AI 时代的超级个体」

关键主题提炼：

| 主题 | 对产品的贡献 |
|------|-------------|
| GMV 拆解（流量 × 转化率 × LTV） | 支撑利润诊断模块的商业诊断框架 |
| 精益验证（先验证获客成本） | 获客成本字段设计：拉新才产生，不获客为零 |
| 竞对分析（AI 调研竞品） | AI 技能维度题目素材 |
| 媒体杠杆 + 个人 IP | 市场和客户关系维度 OPC 内容 |
| Vibe Coding + AI 自动化 | 独立 AI 技能维度（第 11 维） |
| Token 成本意识 | Token（月）独立成本字段 |
| 反脆弱结构（收益无限、亏损有底线） | 抗风险维度 OPC 内容 |

## 三、技术栈裁决（已定稿）

| 层级 | 选型 | 来源 | 理由 |
|------|------|------|------|
| 框架 | **Taro 4.x + React 18** | Meoo 平台 | 微信小程序 weapp 编译 |
| CSS | **Tailwind CSS 4** | Meoo 强制 | 无 SCSS、无第三方 UI 库 |
| UI 组件 | **自建 Tailwind 组件** | Meoo 强制 | 禁止 Element Plus |
| 状态管理 | **Zustand** | Meoo 模板 | 轻量，React 生态 |
| 图表 | **纯内联 SVG** | 自定 | 小程序兼容，体积 ~5KB |
| 高精度计算 | **decimal.js** | A+B | 两份报告唯一共同推荐 |
| 数据存储 | **Taro.StorageSync** | Meoo 强制 | 小程序存储规范 |
| Excel 导出 | **SheetJS mini** | 自定 | ExcelJS 依赖 Node Buffer，不兼容 |
| PDF 导出 | **图片保存 + CSV** | 自定 | 小程序无 window.print() |
| 国际化 | **i18next + react-i18next** | A+B | 两份报告共同推荐 |
| 部署 | **Meoo CDN** | 平台要求 | FREE 计划零成本 |

## 四、命名与设计裁决

| 决策项 | 裁决 | 说明 |
|--------|------|------|
| 产品名 | **一人公司罗盘** | 原 Profit Planner，中文更有记忆点 |
| 核心文案 | "你适合一人公司吗？心里没谱 / 成本门儿清，回本没底？ / 丢个数，我们帮你算明白" | 口语化，创业者一眼对号入座 |
| 主题 | **浅色 + 深蓝面板** | 深空蓝 #162340 + 香槟金 #C5A059 + 暖灰白 |
| Tab 命名 | 首页 / 测评 / **ROI** / 我的 | ROI 简短有力，创业者日常用语 |
| 词汇规范 | "事业"替代"企业"，"创业者"替代"企业主" | 全文统一 |
| 成本字段 | 新增 Token（月）、获客成本；定价/销量标注"可选" | 适配 AI 时代 OPC |
| 可视化 | **多彩图标信息图风格** | 绿色收入线 + 红色成本线 + 金色盈亏平衡点 |
| 导航 | **分页跳转** | 内容密度高，小程序心智是点对点 |
