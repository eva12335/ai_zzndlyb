# PRD：创业者利润计划工具

> 版本: V1.0 | 日期: 2026-06-05 | 状态: 需求定义

---

## 一、产品定位

### 1.1 一句话描述

帮创业者从"我只知道要花多少钱"出发，反向推导"定什么价、卖多少件、能不能回本"的交互式财务探索工具。

### 1.2 目标用户

| 用户类型 | 典型场景 | 痛点 |
|---------|---------|------|
| **产品型创业者** | 开实体店、做电商、摆摊 | 知道房租和进货价，不知道卖多少钱、卖多少件才回本 |
| **服务型/一人公司** | 自由设计师、独立开发者、咨询顾问 | 知道每月固定开销，不知道怎么定服务费、需要几个客户才收支平衡 |

### 1.3 核心差异化

传统工具（LivePlan、Enloop）是"正向填表 → 生成利润表"，用户需要提前知道收入数据。

本工具是"**填入成本 → 反向探索可行方案**"：用户不知道定价和销量没关系，系统帮你推导。

## 二、用户故事

### 2.1 主流程

```
用户小明想开一家写字楼咖啡店。
他知道：
  - 月租金+工资+水电 ≈ ¥20,000
  - 每杯咖啡材料成本 ≈ ¥5.5
他不知道：
  - 每杯该卖多少钱？
  - 一个月能卖多少杯？
  - 能不能回本？

小明打开工具 → 选"产品型" → 填入已知成本 → 系统展示：
  "如果定价 ¥25/杯，每月需卖 1,026 杯才能回本"
  "如果定价 ¥30/杯，每月需卖 816 杯才能回本"
  "如果你想月赚 ¥10,000，定价 ¥30 的话需要卖 1,333 杯/月"

小明拖动滑块，实时看到不同(P, Q)组合下的利润变化，最终做出决策。
```

### 2.2 服务型用户故事

```
小红是独立UI设计师，每月固定开销 ¥5,000，每客户获客成本约 ¥200。
不知道时薪该定多少，需要几个客户。

打开工具 → 选"服务型" → 填入成本 → 系统展示：
  "时薪 ¥200/h，每月需计费 29 小时回本"
  "时薪 ¥300/h，每月需计费 17 小时回本"
  "但你每个月最多可计费 120 小时，收入天花板 ¥36,000"
  "如果采用月费制，¥3,000/月/客户，需要 2 个客户回本"
```

## 四、计算模型

### 4.1 统一公式

```
利润 = (单位收费 U - 单位变动成本 VC) × 业务量 V - 固定成本 FC

产品型：U = 单价，V = 月销量，VC = 材料成本/件
服务型：U = 时薪/客单价，V = 计费小时/客户数，VC = 获客成本
```

### 4.2 核心推导

| 计算目标 | 公式 |
|---------|------|
| 盈亏平衡业务量 | V₀ = FC / (U - VC) |
| 盈亏平衡收入 | R₀ = FC / (1 - VC/U) |
| 目标利润所需业务量 | V = (FC + T) / (U - VC) |
| 目标利润所需定价 | U = VC + (FC + T) / V |

### 4.3 服务型产能约束

```
月最大计费小时 = 120h（默认，可调整）
月收入天花板 = 时薪 × 120
超过天花板 → UI 红色预警
```

### 4.4 三场景系数

| 场景 | 收入系数 | 用途 |
|------|---------|------|
| 保守 | 0.7 | 最坏情况 |
| 中性 | 1.0 | 基准预测 |
| 乐观 | 1.3 | 最好情况 |

## 六、输出报表

### 6.1 利润表（P&L）

| 项目 | 计算来源 |
|------|---------|
| 一、营业收入 | P × Q |
| 减：营业成本 | VC × Q |
| 二、毛利 | 收入 - 成本 |
| 毛利率 | 毛利 / 收入 × 100% |
| 减：固定费用 | FC |
| 减：折旧摊销 | 初始投资 / 分摊月数 |
| 三、营业利润 | 毛利 - 固定费用 - 折旧 |
| 减：税费估算 | 营业利润 × 综合税率 |
| 四、净利润 | 营业利润 - 税费 |
| 净利率 | 净利润 / 收入 × 100% |

### 6.2 现金流量表（按月）

| 项目 | 计算来源 |
|------|---------|
| 期初现金 | 上月期末 |
| + 现金流入 | 回款（含账期调整） |
| - 现金流出 | 成本+费用+采购+税费 |
| = 期末现金 | |
| 预警线 | 期末 < FC → ⚠️ |

### 6.3 盈亏平衡分析图

- X轴：业务量（销量/小时数）
- Y轴：金额
- 总收入线（斜率 = 单价）
- 总成本线（截距 = 固定成本，斜率 = 单位变动成本）
- 交叉点 = 盈亏平衡点
- 服务型附加：产能天花板虚线

## 七、非功能需求

| 需求 | 标准 |
|------|------|
| 计算精度 | 使用 decimal.js，所有金额运算保留 2 位小数 |
| 响应时间 | 滑块操作到图表更新 < 100ms |
| 移动端 | 支持 iOS Safari / Android Chrome，触摸操作友好 |
| 离线可用 | PWA，首次加载后可离线使用（本地计算+IndexedDB） |
| 多语言 | 中/英，使用 i18next，语言文件独立 |
| 部署成本 | ≤ ¥4/月（Vercel 免费托管 + 域名 ¥50/年） |
| 隐私 | 财务数据默认存本地，不上传服务器 |
